# Post-Layout
This directory contains the post-layout artifacts for SIWADO generated from Cadence Innovus place and route and extracted in Magic. The layout was verified DRC clean in Magic, and the extracted netlist was used for switch-level simulation in IRSIM, with results available in `../testbenches/Post-Layout/`.
